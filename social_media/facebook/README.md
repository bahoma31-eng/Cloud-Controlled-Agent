# Facebook Auto Poster

سكريبت Python لمراقبة مجلد `media-pipeline/output` ونشر أي صورة جديدة على صفحة Facebook تملكها/تديرها، ثم نقلها إلى `media-pipeline/processed`.

## المتطلبات
- Python 3.10+
- Page Access Token صالح
- صلاحية النشر على الصفحة

## التثبيت
```bash
pip install -r requirements.txt
