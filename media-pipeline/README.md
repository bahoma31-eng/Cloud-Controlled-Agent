# Media Pipeline

مسار معالجة الصور والوسائط لوكيل التنفيذ السحابي.

## الهيكلة

| المجلد | الوصف |
|--------|-------|
| `input/` | ضع الصور هنا للمعالجة |
| `output/` | النتائج النهائية بالمقاسات المطلوبة (FB/IG/TikTok) |
| `archive/` | نسخة احتياطية من الصور الأصلية قبل التعديل |

## سلوك العمل

1. الوكيل يراقب مجلد `input/`
2. إذا وجد صورة جديدة → يعالجها بالمقاسات المطلوبة
3. يحفظ النتائج في `output/`
4. ينقل الأصل إلى `archive/`
5. يبقى `input/` فارغاً بعد المعالجة
